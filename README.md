# Puzzle Graph Solver 🧩

Цей скрипт вирішує задачу об'єднання розрізнених текстових або числових фрагментів у єдиний найдовший ланцюжок. Програма аналізує збіги між закінченнями та початками фрагментів (по 2 символи) і збирає їх в оригінальну послідовність.

## 🚀 Основний підхід
В основі рішення лежить перетворення задачі на теорію графів:
* **Вузли (Nodes):** Окремі фрагменти тексту.
* **Ребра (Edges):** Направлені зв'язки між фрагментами, де останні 2 символи одного збігаються з першими 2 символами іншого.
* **Алгоритм:** Використовується пошук у глибину (**DFS - Depth-First Search**) для знаходження найдовшого шляху в орієнтованому графі.

## 🛠 Технології
* **Мова:** JavaScript (чистий ES6+, класова архітектура)
* **Середовище:** Node.js
* **Модулі:** вбудований модуль `fs` (File System)

## ⚙️ Встановлення та запуск

1. Переконайтеся, що на вашому комп'ютері встановлено Node.js.
2. Створіть файл `source.txt` у кореневій папці проекту.
3. Додайте в нього ваші фрагменти. Дані можуть бути записані як у стовпчик, так і в один рядок (через пробіл чи табуляцію) — алгоритм автоматично їх розпізнає.
4. Запустіть скрипт командою:

```bash
node index.js
